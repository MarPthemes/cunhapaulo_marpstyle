# Install

As a MARP¬¥s prerequisite, remember to include the path to the theme files (CSS files) in your `workspace.code-workspace` in order to make them available to your slide deck.

## It might look similar to:

<br>

```json
{
	"folders": [
		{
			"path": "."
		}
	],
	"settings": {
		"markdown.marp.themes": [
			"./style/socrates.css"
			"./style/plato.css"
			"./style/descartes.css"
			"./style/leibniz.css"
			"./style/kant.css"
			"./style/hegel.css"
			"./style/freud.css"
			"./style/husserl.css"
			"./style/godel.css"
			"./style/heidegger.css"
			"./style/jobs.css"
			"./style/schema.css"
			"./style/structure.css"
		]
	}
}
```
